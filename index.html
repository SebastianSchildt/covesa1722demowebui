Open1722 + VSS

ACF_CAN

This data is received as a CAN stream packaged into IEEE1722 compliant Ethernet frames. It is then converted into VSS using a KUKSA data provider and displayed using KUKSA's VISS implementation.

ACF_VSS

The µC based zone ECU can provides data in VSS semnatic directly over IEEE1722 using ACF_VSS. It is feed without any further mapping/conversion into KUKSA and displayed using KUKSA's VISS implementation
